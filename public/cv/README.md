# CV Folder

## Instructions

Place your CV PDF file in this folder with the name: `Fabian_Prosper_CV.pdf`

The CV component will automatically load and display your resume from this location.

### Features:
- **Preview Mode**: Users can view your CV in a modal without downloading
- **Download Option**: Users can download your CV as a PDF file
- **Responsive Design**: Works on all devices

### File Requirements:
- Format: PDF
- Filename: `Fabian_Prosper_CV.pdf`
- Recommended size: Under 5MB for optimal loading

### Note:
If you want to change the filename, update the `cvFilePath` variable in `/src/components/CV.tsx`
